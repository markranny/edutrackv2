How to use the template:

  • The first column should contain student names
  • Columns 2-5: Written Works scores (1-4)
  • Columns 9-12: Performance Tasks scores (1-4)
  • Column 16: Quarterly Assessment score
  • All other columns will be calculated automatically
  • Save as Excel format (.xlsx) before importing
